# Movie Library 🎥

A Movie and TV Series Library built with ReactJS and TypeScript using The Movie Database API.

## 🛠️ Technologies

- ReactJS
- TypeScript
- React Router
- Styled Components

## 🌐 API

- [TMDB - The Movie Database API](https://developers.themoviedb.org/3/getting-started/introduction)
